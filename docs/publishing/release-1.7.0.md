# RiserJuices 1.7.0 Release Notes

RiserJuices 1.7.0 is the first full fruit expansion update. It turns the mod into a survival-ready fruit and drink system with natural fruit sources, renewable farming, new drinks, and gameplay effects.

## Highlights

- Added grapes, oranges, green apples, coconuts, pineapples, strawberries, blueberries, bananas, cherries, and mangoes.
- Added natural generation for every fruit source.
- Tree fruits now grow on trees with fruiting leaves.
- Grapes generate as vines and can also be found through village house chests.
- Strawberries and blueberries generate as bushes.
- Pineapples generate as low tropical plants.
- Coconuts generate on coconut palms with their own palm log.
- Bananas generate on banana plants with their own banana stem.
- Added 64x64 textures for fruits, drinks, crops, fruit leaves, palm logs, and banana stems.
- Added drinks with effects: grape juice, wine, orange juice, green apple cider, coconut water, tropical juice, strawberry smoothie, blueberry smoothie, banana smoothie, cherry juice, and mango juice.
- Added proper loot tables, models, blockstates, translations, recipes, and creative tab entries.

## Gameplay

Players can find fruits naturally in fitting biomes, harvest them, collect saplings or plant blocks, and build renewable farms. Drinks are crafted from fruits and provide short useful effects, with balance tradeoffs where needed.

## Requirements

- Minecraft Java Edition 26.1.2
- Fabric Loader 0.19.2
- Fabric API
- Fabric Language Kotlin
- Java 25

## Notes

This update is intended as a release build. Existing worlds need newly generated chunks for the new natural fruit sources to appear.
